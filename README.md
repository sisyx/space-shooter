# AI Powered Space Shooter

this is just a simple game but the fun is that you control the game with your hands in the air and not on the keyboard

## Features
1. hand detection with mediapipe
2. move aircraft and fire using hand tracking

# Installation & Running
create a virtual enviroment and install python 12
also install `mediapipe`, `opencv`, `numpy`, `pandas`
and then run:
```python
python main.py
```
